# kanjivg-writer

Render and write kanji using a canvas, based on stroke data from [KanjiVG](https://github.com/KanjiVG/kanjivg).

# Remixed/derived files

The script under `./bin/convert` reads all KanjiVG files, and writes all the paths into a single
JSON file under `./src/stroke_data.json` (or `./dist/stroke_data.json` for the published version).
